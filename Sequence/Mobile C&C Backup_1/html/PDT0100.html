PRODUCT

HYBRID MOBILE PLATFORM, bizMOB™4.0

  • bizMOB™4.0
  • maru
  • bizPUSH

bizMOB™4.0 플랫폼은 급변하는 모바일 환경에서 고객을 위한 다양한 모바일 서비스를 안정적이고 효율적으로 제공하는 플랫폼 입니다.
서버, 단말기, 레거시 연동, 개발, 관리 및 운영 업무 레이어를 제공하는 하이브리드 방식으로
미래창조과학부 인증기관에서 GS 인증된 공인 제품입니다.

bizMOB™4.0
주요 특징

01

Cross Platform

One Source Multi Use

하나의 개발 소스로 Mobile Hybrid APP(iOS,AOS), Web (Responsive)동시 지원

02

Any System

Whatever OS,DB,WAS

Any OS (AIX, WS, Linux..), Any DB (Oracle, MSSQL..), Any WAS (Tomcat, Jeus..) 지원

03

Security

Covering the Key Security

통신망 암호화, 첨부파일 및 본문 내용 보안, 위변조 및 루팅 체크 등 기본 모바일 보안 기능을 별도 agent 설치 없이 bizMOB 플랫폼 자체 에서 제공

04

Service Builder

Powerful tool for Developer

25년간 수 많은 프로젝트 경험이 녹아있는 전문 설계/관리툴을 통해 편리하고 안정적인 개발 지원

05

Plug-in Adaptor

Optimized Interface Method

수 많은 프로젝트를 통해 생성된 어댑터를 통해 손쉬운 Native 연동 (핸드폰 내부기능 및 외부기기 연동), 고객사 레거시 시스템 연동, 3rd Party SW (문서뷰어/에디터 /MDM 등) 연동

06

Any Server

Server whatever

On-Premise 서버 및 Cloud (AWS, AZURE 등) 서버 환경 동시 제공

07

Mutilingual

Language wherever

다국어 Web/App 제작 지원, 시스템 기능에 대한 다국어 및 확장 언어팩 지원

08

UI 프레임웍 확장

Open UI Framework 확장

Web Framework ( TypeScript/ Vue.js/Angular.js)의 도입으로 독립적인 Front Service 구성가능. 상용 UI Framework 도입으로 개발 생산성, 인터렉티브한 어플리케이션 구현을 위한 UI Performance 최적화 환경 제공, Native Plug-in 연동을 위한 Interface 제공

09

bizMOB Portal

Application Management System

Company/User/User Group 관리, App & Contents 관리, App 관련 통계 현황 기능 제공, 폐쇄형 App Store 제공

10

Legacy Interface

Easy Legacy Connecting

SAP, CRM 등 대부분 유형의 Legacy 시스템 연동 기능 제공

11

Push Service

Effective Communication Tool

관리자 페이지를 통해 원하는 대상을 지정하고 중요한 이벤트나 메시지를 어플리케이션의 Push를 통해 직접 전달 가능

12

Device Management

Close Device Management

관리자 페이지를 통해 사용자별 단말기 관리 및 분실 관리 기능 제공. 단말기 변경, 분실 등의 이유로 사용중지 등록을 한 경우 App 사용불가 형태로 만들고 관련데이터 존재 시 삭제 처리

13

App 배포 및 버전 관리

Total Application Management

관리자 페이지와 앱스토어 서버를 통해 초기 배포부터 자동 업데이트 까지 어플리케이션 픽스 및 업그레이드 패치 가능

주요기능

bizMOB™ 플랫폼의 Device 실행환경, 개발환경, Server Platform, 배포, 관리, 운영 환경에서의 주요 기능을 소개합니다.

단말 실행 환경

bizMOB™ 의 단말 실행환경은 하이브리드 앱 개발 기준으로 Android 및 iOS 를 지원하며, 모든 형태의 UI Layer를 독립적으로 구성하여, 각 OS 별 새로운 컨텐츠를 쉽게 적용할 수 있는 장점이 있습니다.

  • 하이브리드앱기반 Android,IOS 및 반응형 웹 지원
  • One source Multi Page 지원
  • UI Layer 독립적 구성
  • Base Plug-in 및 Custom Plug-in 제공
  • 컨텐츠 실시간
  • 무선 업데이트 기능
  • 루팅 및 앱 위변조 등 보안기능
  • 음영지역 지원 기능 등

개발 환경

bizMOB™ 의 UI Container는 다양한 모바일 OS에 대응하기 위해, One-source Multi-Platform 개념의 Hybrid 방식으로 Client UI를 구현합니다. 이러한 방식은 Web 기술인 HTML, CSS, JavaScript 를 이용하여 하나의 컨텐츠를 다양한 단말에서 동일하게 실행할 수 있어, 개발 비용을 줄이고 유지 보수가 용이한 장점이 있습니다.

  • 서버 개발 생산성 향상 Tool
  • 서비스 정의 Tool
  • 웹 시뮬레이터 Tool
  • Templet 생성 Tool
  • 상용 에디터 플러그인 제공 등

서버플랫폼

bizMOB™ 의 서버 플랫폼은 Multi OS를 기반으로 대부분의 Legacy 시스템과 비즈니스 목적에 따른 솔루션을 효율적으로 연동할 수 있는 구조로 설계되어 시스템 간 확장성과 안정성을 보장합니다.

  • Multi OS 기반
  • Multi RDB
  • 리소스 배포 및 원격 단말제어
  • Legacy 시스템 연동 어댑터
  • 생산성 향상
  • General Asset 제공
  • Cloud 및 In house 환경
  • 실시간 로그 뷰어
  • 멀티로그인 제어기능
  • 구간암호화 기능 등

업무연동 환경

bizMOB™ 플랫폼은 모든 상용/비상용
WAS와 호환되며, OS도 Windows Server,
AIX, Linux 등에 관계없이 동작합니다.
사용 가능한 DB도 Oracle,
MSSQL, DB2 등으로 제한이 없으며,
레거시 시스템과는 RESTful API 방식
Web Service, EWS(Exchange Web Service),
WebSocket 등 다양한 기술로
연동할 수 있습니다.

  • Exchange 어댑터
  • IMAP/POP3 어댑터
  • DB 어댑터
  • SAP 어댑터 등

운영 환경

bizMOB™ 플랫폼은 각종 관리와 운영을 위한 관리자 포털을 단일 관리 포인트로서 제공합니다. 이 사이트를 통해 스마트오피스 단말기 관리, 어플리케이션 관리, 사용자 관리, 사용 통계, 로그 관리 등을 할 수 있습니다.

  • 어플리케이션 버전관리
  • 자동배포
  • 권한관리
  • 단말기 관리
  • 사용자 관리
  • 통계관리 등
