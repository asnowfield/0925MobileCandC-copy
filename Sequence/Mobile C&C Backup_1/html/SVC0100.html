모바일 플랫폼 공급 사업

기업의 다양한 모바일 요구사항에 대응할 수 있는 기업용 모바일 플랫폼을 공급합니다. 모바일 플랫폼은 서버 플랫폼, 단말 실행환경,개발환경, 레거시 업무연동 환경,운영 환경으로 구성되어 있으며 수 많은 고객사로부터 검증된 플랫폼입니다.
Enterprise Omni Platform 모든 서버 환경/디바이스/OS 대상으로 Cross Platform 컨셉의 하나의 개발 소스를 통해 앱/Web 구축이 가능한 플랫폼. Cloud와 On-premise 서버 구축, 다국어지원, 보안성 강화 등을 제공. Core Banking Framework 디지털 뱅킹의 핵심인 코어 뱅킹 시스템을 위한 프레임워크. 은행 내/외부에서 발생하는 모든 유형의 거래 관리,금융 소비자가 이용하는 디지털 뱅킹, 외부 금융 시스템 연동 기능 제공 Intelligent Push Message Solution 지능형 대용량 푸시 메시지 발송 시스템.
수십~수백만 명의 사용자를 대상으로 일괄 동시 전송, 수신자의 타겟 속성에 따른 선택 전송, 컨텐츠 및 시스템 상황 변화 조건에 따라 자동으로 메시지를 발송하는 푸시 시스템
